# 📊 Sales Dashboard (Power BI)

## 🔍 Project Overview
This project analyzes sales data using Power BI to generate insights on revenue, profit, and customer trends.

## 🛠 Tools Used
- Power BI
- Excel

## 📈 Key Insights
- Clothing category has the highest sales
- COD is the most used payment method
- Some months show negative profit trends
